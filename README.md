# Celeritas Extra

This is an unofficial backport of Sodium Extra for Celeritas 12.2 (Cleanroom Loader).
Additionally, several custom enhancements are being implemented.

## Description

Celeritas Extra is a comprehensive unofficial backport of Sodium Extra/Embeddium Extra, specifically targeting Celeritas 1.12.2, along with several additional features.

## Requirements

* **CleanroomLoader**
* **Celeritas**
